### Conditions

* **If user hasn't set up their network yet**
  User is shown the [sync network](/sync-network) modal so they sync their connections first

* **If no path was found to a specific person or company**
  User is shown the [sync network](/sync-network) modal so they can add more accounts

* **If more than one path was found**&#x20;
  The [browse paths](/browse-paths) modal experience will show up.&#x20;
